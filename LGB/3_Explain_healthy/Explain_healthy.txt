Analysis

input:
--train_data train data csv
--model model
--organ string, names the outputs according to the organ of origin of the train data

--ranges: csv file with the columns "Name", "Lower", "Upper", "Venn_x" where "Name" defines the title of the window (e.g. youth, middle_age, 3months etc.) and lower and upper contain the range boundaries. 
Venn_x: A number of venn diagrams can be generated comparing either two or three groups. x: the number of the venn diagram, the entries of the venn_x columns are denoting the groups present in the diagram with "y" or "n"

--num_attributions: number of feature attributions (top k) to extract in a csv file
--num_genes_int: optional, if provided gene-gene interactions will be estimated with the shap.utils.approximate_interactions function. Input: number of genes for which to extract interaction parters, has to be <= to num_attributions
--num_int: optional, number of interaction partners to extract for each gene

output:
waterfall.png           one for each age range
beeswarm.png            one for each age range
top_k_genes_global.csv  one file containtin the top k features considering all 
top_50_genes.csv        one file for each age rane, each containting the top features considering this range
venn.png                venn plots comparing age windows as specified in the ranges.csv
dependence_plot_global.png  plots the top 10 features and its top interactor as estimated by the shap framework
dependence_plot.png        one dependence plot for the top 10 genes of each age range
top_i_interactions.csv  one file for each age range, contains the top i interactors of the top g features, where i and g are provided by the user


The script is used on the train data and calculates shap value as feature importances. Optionally feature-feature- interactions can be estimated (see shap documentation shap.utils.estimate_interactions) if the --num_genes_int flag is provided with an integer value.
